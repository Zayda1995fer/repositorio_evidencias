## 💡 Nombre del Proyecto

Sistema Web de Agendamientos de Citas Médicas mediante Arquitectura Monolítica Modular aplicado a una Clínica Privada

## 📖 Descripción del Proyecto

El proyecto consiste en el desarrollo de una aplicación web para la gestión de citas médicas en consultorios privados. El sistema permitirá al médico administrar pacientes, programar y cancelar citas, consultar agendas diarias y visualizar reportes básicos de atención. La solución busca reducir errores de organización, evitar conflictos de horarios y mejorar la experiencia tanto del profesional de la salud como de sus pacientes mediante herramientas digitales accesibles y de bajo costo.
