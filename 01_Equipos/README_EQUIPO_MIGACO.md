# 📖 Descripción del Proyecto

Miga-Co es una plataforma web SaaS (Software as a Service) orientada a la comercialización de productos de repostería artesanal. El objetivo es darle a cada pastelería suscrita su propio punto de venta digital: catálogo visual, personalización de pedidos y cálculo de envío en tiempo real, todo corriendo sobre una misma infraestructura compartida con aislamiento por tenant.

El sistema utiliza el patrón Shared Database con discriminador de tenant (campo `tenantId`) y combina el modelo Cliente-Servidor con una Arquitectura Limpia por Capas: Presentación, Aplicación, Dominio e Infraestructura.
